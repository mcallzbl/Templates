# Templates Workspace

This workspace uses Copier for project templates.

## Install Copier with pipx

```bash
pipx install copier
```

If this is your first pipx install, make sure the pipx bin dir is on PATH:

```bash
pipx ensurepath
```

Verify:

```bash
copier --version
```

## Basic usage

```bash
copier copy /path/to/template /path/to/new-project
```
